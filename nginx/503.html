Erreur 503

Désolé, le site est temporairement indisponible.

Vous êtes probablement dans un de ces deux cas :

Cause Solution
1er cas De nombreuses personnes au sein du même bâtiment que vous (ou plutôt à la même adresse IP, pour être précis) consultent actuellement le site. Contactez-moi, je modifierai le serveur pour corriger cela. Si vous le savez, indiquez le nombre de personnes utilisant simultanément le site. À titre informatif, j’aimerais aussi savoir ce qui nous vaut l’honneur d’un tel engouement !
2nd cas Vous êtes en train de tenter une attaque par déni de service. Inutile de vous fatiguer, le site est protégé contre ce type d’attaque. Vous voyez cette erreur car nous vous empêchons temporairement d’accéder au site. Cela n’a donc aucun impact sur les autres utilisateurs.

Bertrand Bordage, développeur & administrateur
